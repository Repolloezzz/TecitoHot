<BImp
  title="Definición"
  icon="flag"
  color="orange"
  comment="¿Qué es una variable?"
>
  Una variable es **una** características, atributo o propiedad de un objeto o
  fenómeno que puede ser medido o contado. La misma variable **puede** cambiar
  en el tiempo o en el espacio.
</BImp>
<BComment type="note">
  Una variable se puede definir a partir de los elementos de una
  [población](/matters/Estadistica/ProbabilidadEst/ConceptosGenerales/poblacion)
  o [muestra](/matters/Estadistica/ProbabilidadEst/ConceptosGenerales/muestras),
  siendo que como lo definimos es una característica de un objeto. Por ejemplo,
  la variable **edad** se puede definir a partir de los individuos de una
  población o muestra.
</BComment>
<BComment type="warn">
  Una variable describe una característica de un objeto, y simplemente se limita
  a describir, no a medir. Por ejemplo, la variable **edad** describe que se
  estudia la edad de los individuos de una población o muestra. Entonces, no es
  más que un campo de información que se puede medir.
</BComment>
<BComment type="warn">
  Como se mencionó, una variable **puede** cambiar, pero tambíen puede
  permanecer estático. Por lo tanto, no necesariamente una variable es dinámica
  o que cambia siempre respecto a algo.
</BComment>

Basado en la toda la información disponible de una población o una muestra, se
puede definir una o más variables respecto a una propiedad de interés. La misma
nos indica cual es la información que se desea estudiar.

<BChord title="Ejemplo de variable 🛒" color="yellow" className="mt-3" open>
  Del supermercado Ketal se desea estudiar a sus clientes para hacer un ranking
  de los productos más vendidos. Para ello, se toma una muestra de 100 clientes
  donde definimos las siguientes variables:
  <Balph>
    + - **Nombre**: Nombre del cliente.
    + - **Edad**: Edad del cliente.
    + - **Sexo**: Sexo del cliente.
    + - **Producto**: Producto más comprado.
    + - **Cantidad**: Cantidad de productos comprados.
    + - **Total**: Total gastado en la compras.
  </Balph>
  Entonces, unicamente de estos 100
  clientes se estudiará las variables definidas y así sacar una conclusión

</BChord>

<BChord title="Película más vista 🎞🕶">
  El cine Cinemex desea saber cuál es la película más vista en sus salas de en
  el mes de febrero. Para ello, se toma una muestra de 100 personas que donde se
  estudiaran las siguientes variables:
  <Balph>
    + - **Edad**: Edad de la persona.
    + - **Visto**: Película vista.
    + - **Cantidad**: Cantidad de veces que se vio la película.
  </Balph>
  Entonces, se estudiará la variable **Visto** para saber cuál es la película
  más vista, las demás variables serán para obtener información adicional.

</BChord>
