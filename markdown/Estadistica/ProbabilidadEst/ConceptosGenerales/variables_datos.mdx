<SectionID id="Variables">Variables</SectionID>
<BImp
  title="Definición"
  icon="flag"
  color="orange"
  comment="¿Qué es una variable?"
>
  Una variable es **una** características, atributo o propiedad de un objeto o
  fenómeno que puede ser medido o contado. La misma variable **puede** cambiar
  en el tiempo o en el espacio.
</BImp>
<BComment type="note">
  Una variable se puede definir a partir de los elementos de una población o
  muestra, siendo que como lo definimos es una característica de un objeto. Por
  ejemplo, la variable **edad** se puede definir a partir de los individuos de
  una población o muestra.
</BComment>
<BComment type="warn">
  Una variable describe una característica de un objeto, y simplemente se limita
  a describir, no a medir. Por ejemplo, la variable **edad** describe que se
  estudia la edad de los individuos de una población o muestra. Entonces, no es
  más que un campo de información que se puede medir.
</BComment>
<BComment type="warn">
  Como se mencionó, una variable **puede** cambiar, pero tambíen puede
  permanecer estático. Por lo tanto, no necesariamente una variable es dinámica
  o que cambia siempre respecto a algo.
</BComment>

Basado en la toda la información disponible de una población o una muestra, se
puede definir una o más variables respecto a una propiedad de interés. La misma
nos indica cual es la información que se desea estudiar.

<BChord title="Ejemplo de variable 🛒" color="yellow" className="mt-3" open>

    + - **Total**: Total gastado en la compras.
  </Balph>
  Entonces, unicamente de estos 100
  clientes se estudiará las variables definidas y así sacar una conclusión

</BChord>

<SectionID id="datos">Datos</SectionID>
<BImp title="Definición" icon="flag" color="orange" comment="¿Qué es un dato?">
  Un dato un conjunto de valores que se obtienen de una o más variables, la
  misma puede ser un valor o una categoría.
</BImp>

<BComment type="note">
  Al decir que un dato puede ser un valor o una categoría nos referimos a que
  puede tomar un valor numérico o una categoría. Por ejemplo:
  <Balph>
    + - **Edad**: 20 años.
    + - **Sexo**: Masculino.
    + - **Color**: Rojo.
  </Balph>
  Donde edad es un valor numérico y sexo y color son categorías.

</BComment>
<BComment>
  Una categoría es una etiqueta o un tipo de clasificador que se le da a un
  dato. Por ejemplo, el sexo puede ser masculino o femenino, la misma se puede
  usar para clasificar a los individuos por sexo.
</BComment>
<BComment type="warn">
  Como tal, un dato no es una variable, pero si se puede definir un dato a
  partir de una variable. Por ejemplo:
  <Balph className="grid grid-cols-2">
    <div>
      **Altura**
      <LComment>Variable</LComment>
    </div>
    <div>
      **174 [cm]**
      <LComment>Dato</LComment>
    </div>
  </Balph>
</BComment>

<BImp title="Tipos de datos" color="purple" icon="at">
  Un dato es un conjunto de información, entonces un dato puede ser:
  <Balph>
    **Univariable**: Es un dato que se obtiene de una sola variable. Es decir
    que únicamente toma un valor ó categoria. Tambíen conocido como dato simple.
  </Balph>
  <Balph>
    **Multivariable**: Es un dato que se obtiene de más de una variable. Es
    decir, puede asemejarse a una tabla de datos donde cada fila es un dato y
    cada columna es una variable. Tambíen conocido como dato compuesto.
  </Balph>
</BImp>

<BComment type="warn">
  El dato univariable es el más simple, ya que únicamente se obtiene de una
  variable. Además, solo es un caso particular del dato multivariable.
</BComment>

<BComment type='note'>
  Es necesario recalcar que un dato es un conjunto de información, algo
  asi como una tabla de donde una dato es una fila y las variables son las
  columnas. Por ejemplo:
  <div className='overflow-auto flex justify-center'>
    <table className='table table-normal w-[90%] my-3'>
      <thead>
        <tr>
          <th>Nombre</th>
          <th>Edad</th>
          <th>Empleo</th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <td>Juan</td>
          <td>19</td>
          <td>Estudiante</td>
        </tr>
        <tr>
          <td>Marcos</td>
          <td>42</td>
          <td>Profesor</td>
        </tr>
      </tbody>
    </table>
  </div>
  De la tabla que se muestra, cada columna es una variable y cada fila es un
  solo dato.

</BComment>

Los datos son la fuente de información recopilada de una población o muestra,
que se utilizan para realizar un análisis estadístico. Los datos se obtienen
a partir de las variables definidas.

<BChord title='Ejemplo de dato 💌' className='mt-3' color='red' open>
